Description:

6DMG loader shows how to load motion gestures/characters/words from 6DMG database into the C++ struct.  The HTK exporter and MATLAB exporter are also included.  It is a good start point to wrap 6DMG into other applications.  The code should work on major compilers, but is only tested on VS2008. 

[MotionGesture.db, MotionChar.db, MotionWord.db are included.]
